# 🛠️ Ubiri — Connecter les Talents Africains

![Ubiri Banner](https://placehold.co/1200x400/065F46/FFFFFF?text=UBIRI+-+L'Artisanat+Connecté)

**Ubiri** est une plateforme de mise en relation de nouvelle génération conçue pour digitaliser l'artisanat et les services de proximité en Afrique de l'Ouest. Elle permet aux clients de trouver des experts vérifiés (plombiers, électriciens, mécaniciens, etc.) et aux artisans de développer leur activité grâce à des outils numériques puissants.

## ✨ Fonctionnalités Clés

*   **🔍 Recherche Intelligente & Géolocalisation** : Trouvez des artisans qualifiés près de chez vous grâce à notre carte interactive.
*   **💬 Messagerie Instantanée** : Échangez des messages, des photos, des positions GPS et des messages vocaux en toute sécurité.
*   **🛡️ Paiement Sécurisé (Escrow)** : L'argent est bloqué par la plateforme et versé à l'artisan uniquement une fois les travaux validés par le client.
*   **📸 Fil Communautaire** : Un espace type "Instagram" pour permettre aux artisans de publier leurs réalisations et de gagner en visibilité.
*   **📦 Marketplace** : Boutique intégrée pour l'achat de pièces détachées et d'outillage professionnel.
*   **📱 Mobile-First** : Optimisé pour une utilisation sur smartphone via Capacitor (Android & iOS).

## 🚀 Technologies Utilisées

*   **Frontend** : React.js, Vite, Tailwind CSS (Design Premium & Dark Mode).
*   **Mobile** : Capacitor (Intégration Native & Notifications Push).
*   **Sécurité** : Hachage SHA-256 pour la protection des données utilisateurs.
*   **Backend** : Node.js & SQLite (Architecture légère et performante).
*   **Cartographie** : Leaflet & OpenStreetMap.

## 📦 Installation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/MrOnesim/Ubiri.git
   cd Ubiri
   ```

2. **Installer les dépendances**
   ```bash
   cd ubiri-app
   npm install
   ```

3. **Lancer en mode développement**
   ```bash
   npm run dev
   ```

4. **Compiler pour Mobile (Capacitor)**
   ```bash
   npx cap sync
   npx cap open android # ou ios
   ```
